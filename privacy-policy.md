# Privacy Policy for Handy AI Copilot

Last updated: [2024/11/22]

## Introduction
This Privacy Policy describes how Handy AI Copilot ("we", "our", or "extension") handles user information.

## Data Collection and Storage
- The extension stores API keys and settings locally in your browser
- No personal data is collected, stored, or transmitted to external servers
- All conversations and generated content remain on your device

## API Communication
- The extension communicates with:
  - OpenAI API (for chat features)
  - Replicate API (for image generation)
- API calls are made using user-provided API keys
- No conversation data is stored on our servers

## Data Usage
- API keys are used solely for authenticating API requests
- Settings are stored locally for user convenience
- No analytics or tracking mechanisms are implemented

## Third-Party Services
- OpenAI (for chat features)
- Replicate (for image generation)
- Users should review these services' privacy policies

## User Rights
- All user data is stored locally and under user control
- Users can clear all data by removing the extension

## Updates to Privacy Policy
We may update this privacy policy. Users will be notified of significant changes.
